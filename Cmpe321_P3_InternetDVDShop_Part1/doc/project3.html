Internet DVD Shop

Deadline: 

  • DB Design Document:  05.06.2006 / Monday

  • Deadline and demo of the complete project: 12.06.2006 / Monday

In this web database application project you will design and implement the database and web page of an Internet DVD Shop. In this shop both film and music DVDs are sold. In the database you will store information about the film DVDs (title, producer, country, director, year, music, scenarist, main actors, duration, quantity in stock, price, image of the cover, etc); music DVDs (title, producer, country, singer, year, duration, quantity in stock, image of the cover, etc);customers of the shop (name, email, telephone, address, username, password, etc); orders (dvd ordered, customer who made the order, quantity, order date, ship date, status, and etc.) The Internet DVD Shop should provide the following functionalities:

  • The administrator of the internet shop should be able to log on to the site by providing his user name and password and add/delete/modify DVDs.

  • New customers should register to the internet shop by entering their personal information and providing a username and password.

  • The registered customers should log on to the portal by providing their usernames and passwords in order to purchase DVDs.

  • Registered customers should be able to update their personal information.

  • Registered customers should be able to search for film or music DVDs by providing the title of the film or music album, or the singer of the music album.

  • Registered customers should be able to add the DVDs they choose to their shopping basket. The total cost should be changed accordingly and presented to the customer.

  • Registered customers should be able to delete the DVDs they choose from their shopping basket. The total cost should be changed accordingly and presented to the customer.

  • Registered customers with items in their shopping basket can check out and make a purchase by submitting their credit card information.

  • Registered customers should be able to track the status of their order.

The project consists of two phases. In the first phase you will design and implement the database for this application. You can use a DBMS system of your choice such as MS SQL Server 2000, MySql, and Sybase to implement the database. In the second phase you will design and implement the web application. The specifications of the two phases are as follows:

DB design and implementation phase:

  • Find out the information requirements of the Internet DVD Shop.
    Determine the constraints and domains.
    Determine the entities and relations.                                                                                                                              
  • Identify the properties of the entities and their domains.                                                                                             
  • Determine the identifier of each entity (i.e., find the primary key).                                                                                      
  • Draw the Entity-Relationship (E-R) diagram
  • Decide what the base relations are.
  • Draw the Functional Dependency diagrams.
  • Make sure your relations are in BCNF (4NF also).
  • Decide what the referential integrity constraints are (identify foreign keys).
  • Decide which deletion integrity rules to use (restrict, set to NULL, or cascade).
  • Identify user views.
  • Considering frequent access, come up with a physical database schema.
  • Create the database on a DBMS.
  • Specify DDL statements in SQL (internal, external, conceptual level).
  • Specify the queries (needed for the transactions you determined in step 1) using SQL.
    Write SQL insertion, deletion, modification and select statements (DML statements).
  • Present your work in the DB design report. Submit the hard copy of your report.
  • Submit the script (*.sql if you use MS SQL Server) of the database you have implemented in a CD/diskette.

Web application design and implementation phase:

  • A simple web interface is adequate.
  • You are recommended to use the following technologies:
    •  Java (Servlets, JSP, and JDBC)
    •  Apache Tomcat Server
    •  Visual Studio .NET

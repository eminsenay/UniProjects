In the implemantation of the project, java programming language is used. The compiler is
the sun java compiler which can be obtained from http://java.sun.com/.

To compile:
javac Cmpe321Project2.java DataPage.java HeaderPage.java Record.java RecordOperations.java TableInfo.java TableOperations.java
To run:
java Cmpe321Project2

Notes:

      1. Eclipse project files and the binary files of the project which was automatically
         created by eclipse are also given with the source files. Without any compilation,
         you can run the program with the command given above.
	
      2. HTML version of the javadoc can be found in the doc folder of the project.
